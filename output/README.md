# Output

This folder contains the final reader-facing deliverables.

## PDF

- `pdf/world_cup_ccs_report_zh.pdf` - Chinese report.
- `pdf/world_cup_ccs_report_en.pdf` - English report.

## Figures

`figures/` contains the PNG exhibits used in the report. Files are language-tagged with `_zh` or `_en` when labels differ.

## Tables

`tables/` contains compact result tables and benchmark summaries. Detail-level intermediate tables are intentionally excluded from the public package.
